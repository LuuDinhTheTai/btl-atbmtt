Mật Mã Caesar

Dịch chuyển từng ký tự trong bảng chữ cái theo khóa K

Mã hóa: C = (M + K) mod 26
Giải mã: M = (C − K + 26) mod 26

Khóa được lặp lại cho đủ chiều dài bản rõ

Mã hóa: Ci = (Mi + Ki mod len(K)) mod 26
Giải mã: Mi = (Ci − Ki mod len(K) + 26) mod 26

Tìm Nghịch Đảo Modulo

x = a−1 mod n — Theo định nghĩa & Thuật toán Euclid mở rộng

Định nghĩa: x = a−1 mod n ⟺ (a × x) mod n = 1
Điều kiện: gcd(a, n) = 1
Euclid mở rộng: Tìm x, y sao cho a·x + n·y = gcd(a, n) = 1 → x mod n là nghịch đảo
Phương pháp: Theo Định Nghĩa (Brute Force)

Thuật Toán RSA — Bài Toán 1

An mã hóa M bằng khóa công khai, Ba giải mã C bằng khóa bí mật

Tạo khóa: n = p×q, φ(n) = (p−1)(q−1), d = e⁻¹ mod φ(n)
Mã hóa: C = Me mod n
Giải mã: M' = Cd mod n
a) Khóa công khai PU = {e, n}
b) Khóa bí mật PR = {d, n}
c) An mã hóa: C = Me mod n
d) Ba giải mã: M' = Cd mod n
